主要讲modbusAP模式web控制八路继电器.py这个程序
当程序运行时会开启一个wifi热点
名称为：ESP32S3_Relay
密码：12345678
ModbusRTU配置如下
波特率：9600
通信接口tx=43（开发板的gpio43脚）
       rx=44（开发板的gpio44脚）
       继电器地址：0x0000, 0x0001, 0x0002, 0x0003, 0x0004, 0x0005, 0x0006, 0x0007（标准地址）
       把on与off配置到所有地址
       已经配置了utf-8
       开启一个Web服务器0.0.0.0端口80
       html控制8个地址
使用说明
1.手机连接wifi热点
2.浏览器访问:http://192.168.4.1
